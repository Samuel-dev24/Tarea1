Curso de Bases de Datos 1

Descripción del Curso

El curso de Bases de Datos 1 ofrece una introducción a los principios fundamentales del diseño y manejo de bases de datos. Los estudiantes aprenderán sobre el modelo relacional, el lenguaje SQL, y técnicas de diseño de bases de datos. El curso está diseñado para proporcionar una base sólida en la creación, gestión y consulta de bases de datos, así como en la aplicación de estas habilidades en situaciones prácticas.

Objetivos del Curso

  • Comprender los principios del modelo relacional de bases de datos.
  • Aprender y aplicar el lenguaje SQL para la manipulación de datos.
  • Diseñar y crear bases de datos relacionales eficaces.
  • Realizar consultas avanzadas y optimizar el rendimiento de las bases de datos.

Contenidos del Curso

Unidad 1: Introducción a las Bases de Datos

Conceptos Clave:

  • Definición y tipos de bases de datos.
  • Modelo relacional y su importancia.
  • Conceptos de entidad, atributo y relación.

Puntos Clave:

  • Entender las bases de datos y su clasificación.
  • Reconocer los componentes clave del modelo relacional.
Concepto Definición Ejemplo
Entidad Objeto o cosa sobre la que se almacena información Cliente
Atributo Característica de una entidad Nombre del cliente
Relación Asociación entre entidades Compra

Unidad 2: Diseño de Bases de Datos

Conceptos Clave:

  • Modelo entidad-relación (ER).
  • Normalización y desnormalización.
  • Diagramas ER y sus componentes.

Puntos Clave:

  • Diseñar diagramas entidad-relación para representar bases de datos.
  • Aplicar reglas de normalización para evitar redundancias.
Concepto Definición Ejemplo
Modelo ER Diagrama que muestra entidades y sus relaciones Diagrama de relaciones entre clientes y pedidos
Normalización Proceso de eliminar redundancias y dependencias 1NF, 2NF, 3NF
Desnormalización Introducción de redundancias para optimizar consultas Combinar tablas para mejorar el rendimiento

Unidad 3: Lenguaje SQL

Conceptos Clave:

  • Sentencias básicas: SELECT, INSERT, UPDATE, DELETE.
  • Funciones y operadores SQL.
  • Consultas avanzadas: JOIN, UNION, subconsultas.

Puntos Clave:

  • Escribir y ejecutar consultas SQL básicas y avanzadas.
  • Utilizar funciones y operadores para manipular datos.
Sentencia Descripción Ejemplo
SELECT Selecciona datos de una base de datos SELECT * FROM clientes;
INSERT Inserta nuevos datos en una tabla INSERT INTO clientes (nombre, email) VALUES ('Juan', 'juan@gmail.com');
JOIN Combina filas de dos o más tablas SELECT * FROM clientes JOIN pedidos ON clientes.id = pedidos.cliente_id;
UNION Combina los resultados de dos consultas SELECT nombre FROM clientes UNION SELECT nombre FROM proveedores;

Unidad 4: Administración de Bases de Datos

Conceptos Clave:

  • Creación y gestión de usuarios y permisos.
  • Respaldo y recuperación de bases de datos.
  • Optimización del rendimiento y mantenimiento.

Puntos Clave:

  • Administrar usuarios y permisos de acceso a bases de datos.
  • Implementar estrategias de respaldo y recuperación.
Concepto Descripción Ejemplo
Usuarios Administración de cuentas de acceso CREAR USUARIO 'admin' IDENTIFIED BY 'password';
Respaldo Proceso de guardar una copia de la base de datos BACKUP DATABASE mydb TO DISK = 'backup.bak';
Optimización Mejora del rendimiento de las consultas Uso de índices para acelerar búsquedas

Evaluación

  • Exámenes Parciales: Evaluación de conocimientos teóricos y prácticos.
  • Proyectos y Tareas: Desarrollo de proyectos prácticos y tareas de consulta SQL.
  • Examen Final: Evaluación comprensiva de todo el contenido del curso.

Materiales y Recursos

  • Libro de Texto: "Fundamentos de Bases de Datos" por Abraham Silberschatz, Henry Korth y S. Sudarshan.
  • Software: Sistemas de gestión de bases de datos (MySQL, PostgreSQL, SQLite).
  • Recursos Adicionales: Tutoriales en línea, documentación SQL, ejercicios prácticos.
